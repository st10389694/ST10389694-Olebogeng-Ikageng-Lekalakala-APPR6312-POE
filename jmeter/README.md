This folder contains a JMeter test plan (GiftOfGivers_test_full.jmx) that targets:
- POST /api/claim
- GET /api/claim

Import the JMX into Apache JMeter, adjust host/port to your staging URL, and run.
A sample jtl (jmeter_results.jtl) is included (synthetic) to serve as an example for the test report.
